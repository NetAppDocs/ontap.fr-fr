= Supprimez les deuxièmes clés d'accès S3 avant de revenir de ONTAP 9.19.1.
:allow-uri-read: 


[role="lead"]
Si vous avez effectué une mise à niveau vers ONTAP 9.19.1 et ajouté une deuxième clé d'accès S3 à un utilisateur, ces clés doivent être supprimées avant que vous puissiez revenir à une version antérieure d'ONTAP.

.Étapes
. link:../s3-config/regenerate-access-keys.html#show-s3-access-keys["Afficher les clés d'accès S3"] pour déterminer si des clés d'accès secondaires ont été ajoutées pour des utilisateurs S3.
. link:../s3-config/regenerate-access-keys.html#remove-an-s3-access-key["Retirez les deuxièmes clés d'accès"] pour tous les utilisateurs S3.
