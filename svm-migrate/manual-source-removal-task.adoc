= Supprimer manuellement la SVM ONTAP source après la bascule du client
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


[role="lead"]
Si vous avez effectué la migration SVM avec le nettoyage source désactivé, vous pouvez supprimer le SVM source manuellement une fois la mise en service client terminée.

.Étapes
. Vérifiez que l'état est prêt pour le nettoyage de la source :
+
[source, cli]
----
vserver migrate show
----
. Nettoyez la source :
+
[source, cli]
----
vserver migrate source-cleanup -vserver <SVM_name>
----
