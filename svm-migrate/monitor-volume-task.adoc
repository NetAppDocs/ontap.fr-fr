= Surveiller la migration des volumes ONTAP
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


[role="lead"]
Outre le contrôle de la migration globale du SVM avec `vserver migrate show` Commande, vous pouvez surveiller l'état de migration des volumes que le SVM contient.

.Étapes
. Sur le cluster de destination, vérifiez l'état de la migration des volumes :
+
[source, cli]
----
vserver migrate show-volume
----
