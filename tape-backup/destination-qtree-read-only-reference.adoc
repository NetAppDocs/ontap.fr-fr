= Le qtree de destination est en lecture seule
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


* *Message*
+
`Destination qtree is read-only`

* *Cause*
+
Le qtree vers laquelle la restauration est tentée de lire uniquement.

* * Action corrective*
+
Essayez de restaurer les données à un autre emplacement.
