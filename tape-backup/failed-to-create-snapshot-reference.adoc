= Impossible de créer un snapshot
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


[role="lead"]
* *Message*
+
`Failed to create snapshot`

* *Cause*
+
Le volume contient déjà le nombre maximal de snapshots.

* * Action corrective*
+
Supprimez quelques snapshots, puis réessayez l'opération de sauvegarde.
