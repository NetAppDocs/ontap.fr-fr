= Impossible de charger une nouvelle bande
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


[role="lead"]
* *Message*
+
`Failed to load new tape`

* *Cause*
+
Erreur dans le lecteur de bande ou le support.

* * Action corrective*
+
Remplacez la bande et réessayez l'opération.
