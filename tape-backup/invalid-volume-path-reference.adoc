= Chemin de volume non valide
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


* *Message*
+
`Invalid volume path`

* *Cause*
+
Le volume spécifié pour l'opération de sauvegarde ou de restauration est introuvable.

* * Action corrective*
+
Relancez le travail avec un chemin de volume et un nom de volume valides.
