= Travail interrompu en raison de l'arrêt
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


[role="lead"]
* *Message*
+
`Job aborted due to shutdown`

* *Cause*
+
Le système de stockage est en cours de redémarrage.

* * Action corrective*
+
Relancez le travail après le redémarrage du système de stockage.
