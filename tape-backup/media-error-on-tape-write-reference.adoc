= Erreur de support lors de l'écriture sur bande
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


[role="lead"]
* *Message*
+
`Media error on tape write`

* *Cause*
+
La bande utilisée pour la sauvegarde est endommagée.

* * Action corrective*
+
Remplacez la bande et relancez la procédure de sauvegarde.
