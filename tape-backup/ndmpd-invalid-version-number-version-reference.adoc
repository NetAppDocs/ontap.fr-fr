= ndmpd numéro de version non valide : numéro_version ``
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


* *Message*
+
`ndmpd invalid version number: version_number`

* *Cause*
+
La version NDMP spécifiée n'est pas prise en charge par le système de stockage.

* * Action corrective*
+
Spécifiez NDMP version 4.
