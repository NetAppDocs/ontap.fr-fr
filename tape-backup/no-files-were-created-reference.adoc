= Aucun fichier n'a été créé
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


[role="lead"]
* *Message*
+
`No files were created`

* *Cause*
+
Une tentative de DAR d'annuaire a été effectuée sans activer la fonctionnalité DAR améliorée.

* * Action corrective*
+
Activez la fonctionnalité DAR améliorée et réessayez le DAR.
