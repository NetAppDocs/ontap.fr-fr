= Échec de l'écriture sur bande
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


* *Message*
+
`Tape write failed`

* *Cause*
+
La bande utilisée pour la sauvegarde est endommagée.

* * Action corrective*
+
Remplacez la bande et relancez la procédure de sauvegarde.
