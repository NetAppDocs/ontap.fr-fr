= Bandes hors service
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


* *Message*
+
`Tapes out of order`

* *Cause*
+
La première bande de la séquence de restauration pour l'opération de restauration n'a pas d'en-tête d'image.

* * Action corrective*
+
Chargez la bande avec l'en-tête de l'image et relancez le travail.
