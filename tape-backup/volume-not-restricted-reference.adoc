= Volume non restreint
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


[role="lead"]
* *Message*
+
`Volume not restricted`

* *Cause*
+
Le volume de destination vers lequel les données sont restaurées n'est pas restreint.

* * Action corrective*
+
Limitez le volume et relancez l'opération de restauration.
