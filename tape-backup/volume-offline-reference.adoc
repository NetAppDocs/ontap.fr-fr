= Volume hors ligne
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


[role="lead"]
* *Message*
+
`Volume offline`

* *Cause*
+
Le volume sauvegardé est hors ligne.

* * Action corrective*
+
Mettez le volume en ligne et réessayez la sauvegarde.
